How to import wallet?

UniWallet supports keystore and private key importing right now. You can enter into "Me" page, then click "Manage Wallets", finally click "Import Wallet" button. After that you can follow the hint to finish importing.

How to export wallet?

You can enter into "Me" page, then click "Manage Wallets", select one of wallet you want to export. Right now UniWallet only supports exporting wallet as keystore string.

How to add more tokens in asset list?

You can enter into "Assets" page, then click "+" button at the end of token list, finally switch on/off the corresponding token. ETH is the default added coin. And when a wallet receives any new token, it will be automatically added into the token list.

Does it cost anything to send and receive funds?

UniWallet is completely free to download and use on all platforms. However, the blockchain networks that administer your funds will charge a small processing fee per transaction to ensure that the transaction goes smoothly and to reward miners. UniWallet itself takes no commision.

Can I send a transaction if I don't have any ETH in my wallet?

No, you cannot send any transaction in this situation. Because each transaction should reward miners with some gas fee on the blockchain networks. So if a transaction is failed to sent, the first thing you should check is that whether you have enough ETH to pay the gas fee.
